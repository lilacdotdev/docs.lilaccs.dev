# Welcome to docs.lilaccs.dev

This is your comprehensive documentation platform built with modern web technologies. Here's everything you need to know to get started.

## Features

Our platform includes several powerful features:

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Dark/Light Mode**: Toggle between themes for comfortable reading
- **Advanced Search**: Find content quickly with our search functionality
- **Tag Filtering**: Browse content by categories and tags
- **Code Highlighting**: Beautiful syntax highlighting for code examples

## Navigation

### Sidebar Navigation

The sidebar on the left contains:

1. **Filter Categories**: Browse posts by topic
2. **External Links**: Quick access to related sites
3. **Theme Toggle**: Switch between light and dark modes

### Content Views

Switch between two viewing modes:

- **Card View**: Visual grid layout with images
- **List View**: Compact list format for quick scanning

## Code Examples

Here's a simple JavaScript example:

```javascript
function greetUser(name) {
  return `Hello, ${name}! Welcome to docs.lilaccs.dev`;
}

console.log(greetUser("Developer"));
```

And here's some CSS:

```css
.theme-toggle {
  background: var(--background);

  border: 1px solid var(--border);
  border-radius: 0.5rem;
  transition: all 0.2s ease;
}

.theme-toggle:hover {
  background: var(--accent);
}
```
